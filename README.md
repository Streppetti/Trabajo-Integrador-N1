# StreeShop

StreeShop es una tienda interactiva ambientada en el universo de los videojuegos. 
La consigna era desarrollar un clon de WhatsApp, pero lo pense de otra manera.

## ¿Cómo funciona?

1. Ingresás tu nombre en el login
2. Elegís un vendedor
3. Chateás con él para ver su catálogo y comprar items
4. Los items comprados se guardan en tu mochila y el oro se descuenta

## Vendedores

- **El Buhonero** — Resident Evil 4
- **Terry** — Dragon Quest
- **Boris** — League of Legends
- **Tienda Pokémon** — Pokémon
- **Temmie** — Undertale

## Tecnologías

- React + Vite
- React Router DOM — rutas con parámetros
- Context API — estado global del jugador
- CSS - con estética pixel art
